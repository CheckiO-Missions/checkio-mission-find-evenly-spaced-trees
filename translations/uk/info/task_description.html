Вам потрібно знайти рівномірно розташовані дерева в гаю.

У якості вхідних даних задано розташування дерев ( набір цілих чисел). Вам потрібно знайти і вивести максимальну кількість рівномірно розташованих дерев. Якщо відповідь менша або дорівнює 2, виведіть None.

Приклади: 

assert find_evenly_spaced_trees({0, 2, 4, 7}) == 3
assert find_evenly_spaced_trees({0, 2, 3, 5, 9}) == None
assert find_evenly_spaced_trees({0, 3, 6, 8, 11, 14, 17}) == 4
assert find_evenly_spaced_trees({0, 4, 6, 8, 12, 16, 18}) == 5


Вхідні дані: Розташування дерев ( набір цілих чисел).

Вихідні дані: Максимальна кількість рівномірно розташованих дерев (ціле число або None).

Вхідні дані:

  • 3 ≤ кількість дерев ≤ 80
  • 0 ≤ позиція ≤ 99
