Sie müssen gleichmäßig verteilte Bäume in einem Hain finden.

Sie erhalten die Position der Bäume (eine Reihe von Ganzzahlen) als Eingabewerte. Sie müssen die maximale Anzahl der gleichmäßig verteilten Bäume finden und zurückgeben. Wenn die Antwort kleiner oder gleich 2 ist, geben Sie Keine zurück.

Beispiele: 

assert find_evenly_spaced_trees({0, 2, 4, 7}) == 3
assert find_evenly_spaced_trees({0, 2, 3, 5, 9}) == None
assert find_evenly_spaced_trees({0, 3, 6, 8, 11, 14, 17}) == 4
assert find_evenly_spaced_trees({0, 4, 6, 8, 12, 16, 18}) == 5


Eingabe: Die Position der Bäume (eine Reihe von Ganzzahlen).

Ausgabe: Die maximale Anzahl der gleichmäßig verteilten Bäume (eine ganze Zahl oder keine).

Vorbedingung:

  • 3 ≤ Anzahl der Bäume ≤ 80
  • 0 ≤ Position ≤ 99
